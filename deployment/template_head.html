Downloads

Wahay is in an early stage of development. Not all of its functionalities have been developed and you may find errors!

Download it at your own risk!

At the moment it's available only for Linux 64 bits, in the future we plan to provide binaries for other operating systems.

Linux downloads

In order for Wahay to work Mumble and Tor should be installed in the operating system. The binary file would work on most Linux distributions, but you would have to install Mumble and Tor (>= 0.3.5) package by your self.

The Bundle packages are a compress directory that includes Wahay binary, Mumble and Tor. They are dependent of the distribution, but there is no need to install in the system. Just download, uncompress and double click on wahay file.

The Linux packages, are packages that would work on specific Linux distribution. At the moment we have deb packages that can be installed in Debian base distributions. It is important to note that the distribution must have Tor >= 0.3.5, that is why it would not work out of the box in Ubuntu 18.04. To fix this issue install a newer version of Tor (https://2019.www.torproject.org/docs/debian.html.en#ubuntu). A simpler solution would be to use the Bundle package if available.

Different options to install Wahay are available. The Binary file is Wahay application without Mumble and Tor.
